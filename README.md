## CGV 도우미

CGV 도우미 앱은 CGV, 롯데시네마 이용자들을 위한 보조 도구로, 공식적인 제휴 관계가 없습니다. CGV 도우미 앱을 통해 다음과 같은 기능을 사용할 수 있습니다:

- **극장별 특전 수량 확인**: 원하는 극장에서 제공되는 특전의 남은 수량을 실시간으로 확인할 수 있습니다.
- **이벤트 페이지 검색 기능**: 다양한 이벤트 페이지를 검색하고 참여할 수 있습니다.
- **스피드쿠폰 취소표 알림**: 스피드쿠폰의 취소표가 발생할 때 알림을 받을 수 있습니다.
- **서프라이즈 쿠폰 취소표 알림**: 서프라이즈 쿠폰의 취소표가 발생할 때 알림을 받을 수 있습니다.
- **영화 예매 오픈 알림**: 관심 있는 영화의 예매 오픈 시 알림을 받을 수 있습니다.
- **반응형 UI 지원**: 다양한 디바이스에서 최적화된 화면으로 이용할 수 있습니다.

## 지원 운영 체제
- Windows  
  .net 8.0 런타임이 필요합니다.

- Android  
  Android 12(API Level 31) 이상

- IOS  
  IOS 17 이상

- Mac Catalyst

CGV 도우미 앱은 CGV의 공식적인 애플리케이션이 아니며, CGV와 별도의 연관성이 없습니다. 이 애플리케이션은 CGV 이용자들을 위한 보조 도구로 제공됩니다.
