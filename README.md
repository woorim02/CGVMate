## CGV 도우미
[![Hits](https://hits.seeyoufarm.com/api/count/incr/badge.svg?url=https%3A%2F%2Fgithub.com%2Fwoorim02%2FCGVMate&count_bg=%2379C83D&title_bg=%23555555&icon=&icon_color=%23E7E7E7&title=hits&edge_flat=false)](https://hits.seeyoufarm.com)

CGV 도우미 앱은 CGV, 롯데시네마 이용자들을 위한 보조 도구로, 공식적인 제휴 관계가 없습니다. CGV 도우미 앱을 통해 다음과 같은 기능을 사용할 수 있습니다:

- **극장별 특전 수량 확인**
- **이벤트 페이지 검색 기능**
- **스피드쿠폰 취소표 알림**
- **서프라이즈 쿠폰 취소표 알림**
- **영화 예매 오픈 알림**
- **반응형 UI 지원**

## 지원 운영 체제
- Windows  
  .net 8.0 런타임이 필요합니다.

- Android  
  Android 12(API Level 31) 이상

- IOS  
  IOS 17 이상

- Mac Catalyst

CGV 도우미 앱은 CGV의 공식적인 애플리케이션이 아니며, CGV와 별도의 연관성이 없습니다. 이 애플리케이션은 CGV 이용자들을 위한 보조 도구로 제공됩니다.
